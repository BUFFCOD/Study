# RHCSA Comprehensive Study & Hands-On Practice Guide

This guide transforms your uploaded notes into a structured, exam-focused roadmap with real hands-on labs. Treat this like your personal RHCSA bootcamp.

## 1. User Management & Default Settings

### Concepts to Master

* /etc/login.defs controls password aging and defaults
* /etc/skel defines default user environment
* Account locking vs expiration
* Service accounts vs human accounts

### Hands-On Labs

**Lab 1: Create Secure User Template**

1. Edit /etc/skel/.bashrc and add a custom banner.
2. Create user:

```bash
useradd labuser1
passwd labuser1
```

3. Verify home content came from /etc/skel.

**Lab 2: Enforce Password Policy**

1. Edit /etc/login.defs
2. Set:

* PASS_MAX_DAYS 30
* PASS_MIN_LEN 10

3. Create new user and verify with:

```bash
chage -l newuser
```

**Lab 3: Lock & Expire Accounts**

```bash
usermod -L labuser1
usermod -e 2025-01-01 labuser1
```

Test login failure.

## 2. Group Management & Access Control

### Skills

* Primary vs secondary groups
* Temporary group switching
* Permission testing

### Hands-On

**Lab 4: Shared Finance Area**

```bash
groupadd finance
mkdir /finance
chgrp finance /finance
chmod 2770 /finance
```

Add users and verify group inheritance.

**Lab 5: Group Switch Testing**

```bash
newgrp finance
touch testfile
ls -l testfile
```

Confirm group ownership.

## 3. File Permissions & Special Bits

### Skills

### Hands-On

**Lab 6: Permission Simulation**

1. Create 3 users.


3. Try delete protections.

**Lab 7: SUID Demo**

```bash

```

Verify privilege behavior.
