
the rpm command can used to list packages and analyze packages contents.

some commands query the rpm database:

```bash
rpm -qa show all installed packages
rpm -qi <package_name> show detailed info about a package
rpm -ql <package_name> list files installed by a package
rpm -qf <file_path> find which package installed a file
rpm -q --scripts <package_name> show scripts associated with a package
rpm -q --changelog <package_name> show changelog of a package
rpm -qc <package_name> list configuration files of a package
```
querying package files is also possible with:

add -p  any of the commands above to query a package file without installing it:

```bash
rpm -qpl <package_file.rpm> list files in a package file
rpm -qp --scripts <package_file.rpm> show scripts in a package file
```

Contents of an RPM package can be extracted without installing it using the `rpm2cpio` and `cpio` commands:

```bash
rpm2cpio <package_file.rpm> | cpio -tv
rpm2cpio <package_file.rpm> | cpio -idmv
```
