## Scheduling Deferred User Jobs

the atd serveice must be running to run once-only jobs using at

Use at<time> to schedule a job at a specific time

type one or more job specifications,  in the interactive prompt.
used ctrl-d to end the job specification.

use atq to list pending jobs
use atrm <job number> to remove a pending job

## Understanding Temporary  Files

In the past temporary files were often stored in /tmp directory.

without management these files could stay around for a long time and consume disk space.

As a solution the /tmp directory could be created on the ram drive

nowadays systemd-tmpfiles is used to manage temporary files and directories.

Functionality has been expanded to manage other files also.

It will create, delete tmp file automatically according to configuration files int the following locations:
/etc/tmpfiles.d/*.conf
/run/tmpfiles.d/*.conf
/usr/lib/tmpfiles.d/*.conf

systemd-tmpfiles works with related services to manage temporary files.
systemd-tmpfiles-setup.service creates and removes temporary files according to configuration. 
systemd-tmpfiles-clean.service removes old files from temporary directories.
by default this happen 15 minutes after boot and daily afterwards.
