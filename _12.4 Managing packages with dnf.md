dnf was created to be intuitive 
 dnf list lists  installed and available packages
    dnf list <package_name> lists information about a specific package
    dnf search <keyword> searches for packages matching the keyword
    dnf search all <keyword> searches for packages matching the keyword in all fields
    
    dnf provides <file_name> shows which package provides the specified file
    dnf info <package_name> shows detailed information about a specific package

    example:
    ```bash
    dnf list
    dnf list vim
    dnf search editor
