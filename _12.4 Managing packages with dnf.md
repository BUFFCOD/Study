dnf quick reference (RHCSA scope)

- Search/list/info:
  - `dnf list` (all enabled repos), `dnf list <pkg>`, `dnf search <keyword>`, `dnf provides <path>`, `dnf info <pkg>`.

- Install/remove/update:
  - `dnf install <pkg>`
  - `dnf remove <pkg>`
  - `dnf check-update`
  - `dnf update`

- Repos:
  - Enable/disable: `dnf config-manager --enable <repo>` / `--disable <repo>`
  - Show: `dnf repolist`

- Module streams (RHEL 8/9):
  - `dnf module list`
  - `dnf module enable nodejs:18`
  - `dnf module reset nodejs`

- Local RPMs:
  - `dnf install /path/to/package.rpm` (resolves deps from enabled repos)

- History/rollback:
  - `dnf history`
  - `dnf history info <ID>`
  - `dnf history undo <ID>`

- Clean caches (after repo changes): `dnf clean all`

Example one-liners:

```bash
dnf search editor
dnf provides /usr/bin/ssh
dnf install vim-enhanced
dnf remove httpd
dnf module enable php:8.2
dnf repolist
```
